    disallowCurlyBraces

    Disallows curly braces after statements.

    Types: Array or Boolean

    Values: Array of quoted keywords or true to disallow curly braces after the following keywords:

    Example

    "disallowCurlyBraces": [
    "if",
    "else",
    "while",
    "for",
    "do",
    "with"
]
    Valid
    if (x) x++;
    Invalid
    if (x) {
    x++;
}
