    disallowSpaceAfterKeywords

    Disallows space after keyword.

    Types: Array or Boolean

    Values: Array of quoted keywords or true to disallow spaces after all possible keywords.

    Example

    "disallowSpaceAfterKeywords": [
    "if",
    "else",
    "for",
    "while",
    "do",
    "switch",
    "try",
    "catch"
]
    Valid
    if(x > y) {
    y++;
}
    Invalid
    if (x > y) {
    y++;
}
