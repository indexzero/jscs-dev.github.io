    disallowSemicolons

    Disallows lines from ending in a semicolon.

    Type: Boolean

    Value: true

    Example

    "disallowSemicolons": true
    Valid
    var a = 1
;[b].forEach(c)
    Invalid
    var a = 1;
[b].forEach(c);
