    disallowSpaceAfterLineComment

    Requires that a line comment (//) not be followed by a space.

    Type: Boolean

    Value: true

    Example

    "disallowSpaceAfterLineComment": true
    Valid
    //A comment
/* A comment*/
    Invalid
    // A comment
